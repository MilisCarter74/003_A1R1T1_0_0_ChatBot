# ChatBot

## Description

**ChatBot** is a simple Python console chatbot that can recognize greetings, solve basic math tasks described in English, and say goodbye. The bot uses keyword recognition to understand user intent and responds with random pre-defined phrases.

---

## Features

- Recognizes greetings and says hello
- Solves simple math operations (add, subtract, multiply, divide) when described in English with numbers
- Handles goodbye phrases and ends the conversation politely
- Responds to unrecognized messages with a default answer

---

## Requirements

- Python 3.8+

## Usage

Run the chatbot in your terminal:
```bash
python chat_bot.py
```
Type your message and press Enter. The bot will respond. To stop, type a goodbye phrase (e.g. 'bye') or use Ctrl+C.

---

## Troubleshooting

- **Bot always answers "Sorry, I didn't understand this message."**
    - Make sure your message includes one of the keywords for greetings, math, or goodbye.
